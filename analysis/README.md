# analysis/ 目录说明

这里放 Claude 产出的所有文件（转写、摘要等）。**所有文件一视同仁，你想改哪个直接改，不用分类、不用复制**。

## 文件命名

- `*_说话人分离.{md,txt}` —— 转写（带说话人分离）
- `*_raw.json` —— 转写原始 JSON
- `*_需求摘要.md` —— 摘要

## 安全网

项目根 `git init` 已开。Claude 重新生成任何文件前会先 `git status` 看一眼：

- 你**没有未提交改动** → 直接覆盖 + 新 commit
- 你**有未提交改动** → 停下来问你"先帮你 commit 当前版本，还是直接覆盖？"

万一覆盖了 → `git checkout HEAD~1 -- <file>` 一秒找回。
