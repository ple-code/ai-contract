## 三、待办 / 后续

### 1. passlib 迁移（计划下轮）
passlib 已停止维护，pin bcrypt 4.0.1 是临时方案。迁移到直接用 `bcrypt` 库改动很小（`security.py` 约 10 行）：
```python
import bcrypt
def hash_password(p): return bcrypt.hashpw(p.encode(), bcrypt.gensalt()).decode()
def verify_password(p, h): return bcrypt.checkpw(p.encode(), h.encode())
```
- **无密码迁移问题**：旧 hash 由 bcrypt 生成，`bcrypt.checkpw` 直接兼容

### 2. paddleocr / paddlepaddle ✅ 已确认保留
