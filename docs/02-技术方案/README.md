# 技术方案/

## 放什么

- **架构设计**：系统总览图、模块划分、数据流
- **技术选型**：选型对比（前端框架、后端、数据库、AI 模型）
- **API 设计**：接口文档、OpenAPI spec
- **数据模型**：ER 图、表结构、字典
- **AI 方案**：模型选型、Prompt 设计、RAG / Agent 架构（本项目重点）

## 写作要求

- 关键决策必须给**理由 + 备选方案**，不要只写结论
- 图优先：用 Mermaid / excalidraw / 飞书画图，文档里嵌链接或代码块
- AI 相关方案要写：模型、上下文窗口、token 成本估算、降级策略
