## 1. 代码拉取

```bash
git clone <repo-url> ai-contract
cd ai-contract
```

## 2. 环境变量

复制 `.env.example` 为 `.env`，填入：

```bash
cp .env.example .env
# 编辑 .env，填写：
# - OPENAI_API_KEY=...
# - DATABASE_URL=...
# - ...
```

## 3. 依赖安装

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
